# raycaster

Simple raycaster game in C++ using [raylib](https://www.raylib.com/)

## Controls

|     Key | Action                  |
| ------: | ----------------------- |
|     `W` | Move forward            |
|     `S` | Move Backward           |
|     `A` | Rotate clockwise        |
|     `D` | Rotate counterclockwise |
| `<esc>` | Quit game               |
